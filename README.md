# Stayfinder 🏠✨

Stayfinder is a C# desktop-based application designed to simplify the process of finding, booking, and managing accommodations.  
It connects **customers**, **hosts (property owners)**, and a **super admin** within one ecosystem.

## 📌 Features

### For Customers
- Secure **Sign Up & Login**
- Search & filter accommodations by:
  - Location  
  - Price range  
  - Availability  
  - Amenities  
  - Ratings  
- View property details with images & reviews
- Add to cart, book, and make secure payments
- Access booking history
- Get seasonal offers & discounts
- Rate & review hosts after stay

### For Hosts (Property Owners)
- Register and manage multiple properties (houses/hotels)
- Add, edit, and remove listings with pricing, availability, and amenities
- View and manage booking requests
- Track earnings and booking history with analytics
- Respond to customer reviews
- Update property availability in real time

### For Super Admin
- Manage all users (customers & hosts)
- Approve or reject property listings
- Monitor and moderate reviews
- Suspend or remove hosts if necessary
- Collect **10% commission** from each booking
- Manage promotions, offers, and discounts
- View platform-wide analytics (users, bookings, revenue)

---

## 🗄️ Database Schema

- **Users** (customers & hosts)  
- **Properties** (houses & hotels)  
- **Bookings** (reservation records)  
- **Payments** (secure transactions with commission tracking)  
- **Reviews** (ratings & feedback)  
- **Cart** (temporary selection of properties)  
- **Notifications** (alerts to users & hosts)  
- **Offers** (discounts & seasonal packages)

📌 Includes **ER Diagram & SQL Queries** for all tables.

## 🚀 How It Works
1. Customers search for stays → filter → add to cart → confirm booking → make payment.  
2. Hosts manage their properties & track revenue.  
3. Super Admin monitors everything, ensuring smooth operation and platform revenue.
